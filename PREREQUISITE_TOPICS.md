# Python Topics Assumed as Prerequisites

These are topics I assumed you already know at a medium level. Review this list and let me know which ones you'd like me to add as exercises to the learning plan.

## Foundational Topics (Not Included)

1. **Basic Object-Oriented Programming**
   - Classes and instances
   - Instance methods, class methods, static methods
   - Inheritance (single and multiple)
   - `super()` usage
   - Special methods (`__init__`, `__str__`, `__repr__`)

2. **Exception Handling Basics**
   - try/except/finally blocks
   - Raising exceptions
   - Custom exception classes
   - Exception hierarchy
   - `else` clause in exception handling

3. **List/Dict/Set Comprehensions**
   - Basic list comprehensions
   - Dict comprehensions
   - Set comprehensions
   - Nested comprehensions
   - Conditional comprehensions

4. **Lambda Functions and Basic Functional Concepts**
   - Lambda syntax
   - Basic map(), filter(), reduce()
   - sorted() with key functions
   - any() and all()

5. **File I/O Operations**
   - Reading and writing files
   - Different file modes
   - CSV file handling
   - JSON file handling
   - Working with file paths (os.path, pathlib basics)

6. **Standard Library Essentials**
   - collections (Counter, defaultdict, deque, namedtuple)
   - datetime and timedelta
   - re (regular expressions basics)
   - os and sys modules
   - argparse for CLI arguments
   - logging basics

7. **String Manipulation**
   - String methods (split, join, strip, replace, etc.)
   - String formatting (f-strings, .format(), %)
   - String slicing
   - Working with encodings

8. **Basic Data Structures**
   - Lists, tuples, sets, dictionaries
   - When to use which structure
   - Stack and queue implementations using lists/deque
   - Dictionary methods and operations

9. **Module and Package Basics**
   - Creating modules
   - Importing (import, from...import, as)
   - `__name__ == "__main__"`
   - Package structure with `__init__.py`
   - Relative vs absolute imports

10. **Virtual Environments and Dependencies**
    - venv/virtualenv usage
    - pip basics
    - requirements.txt
    - Installing packages

11. **Basic NumPy Usage**
    - Creating arrays
    - Basic indexing and slicing
    - Common operations (sum, mean, reshape)
    - Basic matrix operations

12. **Basic Pandas Usage**
    - DataFrames and Series
    - Reading CSV/Excel files
    - Basic data manipulation (selecting, filtering, groupby)
    - Handling missing data

13. **Basic Testing**
    - Writing simple unit tests with unittest or pytest
    - Assertions
    - Test structure and organization
    - Running tests

14. **Scoping and Namespaces**
    - LEGB rule (Local, Enclosing, Global, Built-in)
    - global and nonlocal keywords
    - Module-level vs function-level scope

15. **Basic Type Hints**
    - Primitive type annotations (int, str, float, bool)
    - List[T], Dict[K, V], Tuple, Set
    - Optional[T]
    - Basic function annotations

16. **Argument Handling**
    - *args and **kwargs
    - Keyword-only arguments
    - Positional-only arguments (Python 3.8+)
    - Default argument values

17. **Common Built-in Functions**
    - len, range, enumerate, zip
    - isinstance, issubclass, type
    - getattr, setattr, hasattr
    - min, max, sum

18. **Basic Regular Expressions**
    - Pattern matching
    - re.search, re.match, re.findall
    - Groups and capturing
    - Common patterns

19. **Working with Dates and Times**
    - datetime objects
    - Parsing and formatting dates
    - Timedeltas
    - Timezone awareness basics

20. **Dictionary and Set Advanced Features**
    - dict.get() with defaults
    - dict.setdefault()
    - Set operations (union, intersection, difference)
    - Dictionary merging

21. **Pythonic Idioms**
    - EAFP vs LBYL (Easier to Ask Forgiveness vs Look Before You Leap)
    - Duck typing
    - PEP 8 style guidelines
    - Zen of Python principles

22. **Basic Logging**
    - Setting up loggers
    - Log levels (DEBUG, INFO, WARNING, ERROR, CRITICAL)
    - Basic configuration
    - Logging to files

23. **Command Line Interfaces**
    - sys.argv
    - argparse basics
    - Creating simple CLI tools

24. **Working with JSON**
    - json.loads() and json.dumps()
    - Reading/writing JSON files
    - Handling custom objects

25. **Basic HTTP and APIs**
    - Using requests library
    - GET/POST requests
    - Handling JSON responses
    - Basic API authentication

26. **Environment Variables**
    - os.environ
    - Reading .env files
    - python-dotenv basics

27. **Basic Serialization**
    - pickle basics
    - JSON serialization
    - When to use which format

28. **List/String Slicing Advanced**
    - Step slicing
    - Negative indices
    - Slice objects

29. **Sorting and Ordering**
    - sorted() vs .sort()
    - Custom sort keys
    - Reverse sorting
    - Multiple sort criteria

30. **Basic Algorithms Understanding**
    - Big O notation basics
    - Understanding time/space complexity
    - Common algorithmic patterns
